# sym-ui Brand

개인 사이드 프로젝트 전용 디자인시스템. 따뜻하고 둥글며 블루 중심의 톤.

## 브랜드 보이스
- 간결: 불필요한 장식 없음
- 친근: 부드러운 반경, 웜 그레이
- 신뢰: 블루 프라이머리, 명확한 상태 컬러

## 사용 규칙
- 프라이머리 블루는 CTA 요소에만 사용 (primary button, active state, link)
- 뉴트럴 컬러는 surface / border / text 계층 구분 용도
- 시맨틱 컬러는 상태 전달 목적 외 사용 금지
