# sym-ui Brand

개인 사이드 프로젝트 전용 디자인 시스템. 따뜻하고 둥글며 차분한 indigo 중심의 톤.

## 브랜드 보이스
- 간결: 불필요한 장식 없음
- 친근: 부드러운 반경, 웜 그레이 surface
- 신뢰: indigo 프라이머리, 명확한 상태 컬러

## 사용 규칙
- 프라이머리 indigo 는 CTA 요소에만 사용 (primary button, active state, link, focus ring)
- 뉴트럴 (warm gray) 컬러는 surface / border / text 계층 구분 용도
- 시맨틱 컬러 (success / warning / danger / info) 는 상태 전달 목적 외 사용 금지
- 다크 모드 배경은 절대 밝기 L >= 15% 를 보장 (검정 회피)

## 변경 이력
- v0.3.0: Primary 색조를 Tailwind blue 에서 indigo (light: #4f46e5 / dark: #818cf8) 로 변경
  하여 brand 식별력 보강. shadcn/ui 디폴트 톤에서 차별화하기 위한 결정.
